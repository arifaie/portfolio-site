# Portfolio Website

Personal portfolio website built with HTML, CSS, and JavaScript, showcasing my web development projects and skills.

## Live Site

[View Live Portfolio](https://arifaie.github.io/portfolio-site/)

## Features

- Typing animation effect on hero section
- Sticky navigation bar on scroll
- Smooth scroll to each section
- Active nav link highlight while scrolling
- Contact form with JavaScript validation
- Dynamic copyright year
- Responsive design for mobile and desktop
- About section with profile photo
- Project showcase with links to GitHub repositories
- Clean, professional styling

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages for hosting
